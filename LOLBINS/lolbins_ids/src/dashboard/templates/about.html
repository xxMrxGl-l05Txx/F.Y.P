About LOLBins IDS v1.0.0

System Overview

LOLBins IDS is a behavior-based Intrusion Detection System specifically designed to detect and monitor Living Off The Land Binary (LOLBin) attacks. These attacks abuse legitimate Windows binaries to execute malicious code, making them difficult to detect with traditional signature-based approaches.

This system uses Wazuh as its foundation and extends it with custom detection rules, decoders, and behavioral analysis capabilities to identify suspicious usage patterns of legitimate Windows binaries commonly exploited by attackers.

Key Features
Behavioral Analysis

Detects suspicious patterns rather than just signatures, capturing novel attack techniques.

Custom LOLBin Rules

Specialized detection rules for commonly abused Windows binaries like PowerShell, CertUtil, and Regsvr32.

Real-time Alerts

Immediate notification system with configurable channels and priority-based routing.

Comprehensive Dashboard

Visualize security events, analyze trends, and investigate incidents from a central console.

MITRE ATT&CK Coverage

LOLBins IDS provides detection capabilities for the following MITRE ATT&CK techniques commonly associated with LOLBin abuse:

Execution
  • T1059.001PowerShell
  • T1059.003Windows Command Shell
  • T1059.005Visual Basic
  • T1059.007JavaScript
Defense Evasion
  • T1218.001Compiled HTML File
  • T1218.005Mshta
  • T1218.010Regsvr32
  • T1140Deobfuscate/Decode Files
Command and Control
  • T1105Ingress Tool Transfer
  • T1071.001Web Protocols
  • T1219Remote Access Software
